SensIDL - Downloads

On this page we will provide several links for downloading the SensIDL Toolchain as well as links to released versions as soon as they are available.

Update Site

The SensIDL Toolchain can be downloaded at the following Eclipse Update Site.
Requirements for using SensIDL are Eclipse Mars 4.5 with Eclipse Modelling Tools package and Java 8.
  • In Eclipse workbench, please go to Help and choose Install New Software. With Add... button, please give the link of the update site given above. After that you can choose from Work With dropdown list SensIDL.
  • Please choose every available Feature from the list, if there are more.
  • Finally, please follow the instructions given by Eclipse to complete the installation.
  • Now you are ready to use SensIDL Toolchain. For further informations, you can follow our Documentation page.
