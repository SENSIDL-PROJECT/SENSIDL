SENSIDL

The SensIDL (Sensor Interface Description Language) development framework targets sensor developers as well as the developers of IT-systems connected to these sensors and processing the collected data. It is an open source development framework based on a model-driven software development (MDSD) approach supporting the sensor developers in specifying and implementing the communication interfaces.

Sensidl is maintained by SENSIDL-PROJECT
